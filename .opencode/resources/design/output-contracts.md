# Output Contracts

各スキルの出力最低要件です。

## design-direction

- Goal
- Target users or page role
- Visual direction
- Layout guidance
- Typography guidance
- Color guidance
- Risks or open questions

## design-review

- Findings ordered by severity
- Why each issue matters
- Concrete change proposal
- Quick wins
- Structural follow-ups

## design-implementation-brief

- Page or component scope
- UI sections
- States and edge cases
- Responsive rules
- Suggested component boundaries
- Suggested implementation order
